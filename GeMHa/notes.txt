To run as stand-alone jar...
Have saved all GeMHa Project classes (plus supporting project classes) to one jar file...
D:\JavaExecs\GeMHa.jar
and added to CLASSPATH env variable


To run without assembling all jars into one super application jar...
Because I've included exec-maven-plugin in the POM file, I can execute from maven...
(Change the arguments element specifying the applic settings file to run with.)

mvn exec:java

Alternatively, remove the exec-maven-plugin and exec like this...

mvn exec:java -Dexec.mainClass="gemha.GemhaBatch" -Dexec.args="C:\Users\wadel\git\GeMHa\GeMHa\ApplicSettingsFiles\Applic_FtoFtoF.xml"
